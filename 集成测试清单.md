## 附录：测试脚本

### 一键测试脚本

保存为 `test-integration.sh`：

```bash
#!/bin/bash
set -e

VISUAL_SERVER="http://localhost:3000"
TEST_PROJECT="/path/to/test-project"
WORKSPACE_ID="test-project"

echo "=== 开始集成测试 ==="

# 1. 检查 visual 服务
echo "1. 检查 visual 服务..."
curl -f "$VISUAL_SERVER/api/health" || exit 1

# 2. 执行 Collector
echo "2. 执行 Collector..."
cd "$TEST_PROJECT"
npx /path/to/engineered-spec-visual/src/collector/cli.ts \
  --workspace-id "$WORKSPACE_ID" \
  --server "$VISUAL_SERVER" \
  --json > collector-result.json

# 3. 验证 Collector 结果
echo "3. 验证 Collector 结果..."
INSERTED=$(jq -r '.ingest.inserted' collector-result.json)
if [ "$INSERTED" -gt 0 ]; then
  echo "✅ Collector 上报成功: $INSERTED 条事件"
else
  echo "❌ Collector 上报失败"
  exit 1
fi

# 4. 启动新 run
echo "4. 启动新 run..."
npx ai-spec-auto protocol-step --user-input "集成测试" || exit 1

# 5. 等待推送
echo "5. 等待推送..."
sleep 5

# 6. 验证控制台数据
echo "6. 验证控制台数据..."
curl -f "$VISUAL_SERVER/api/workspaces" || exit 1

echo ""
echo "=== 集成测试完成 ==="
```

执行：

```bash
chmod +x test-integration.sh
./test-integration.sh
```
