# 项目结构（Spring Boot）

> 本文件由 `project-init` 技能根据仓库实际情况生成，请勿手动覆盖自动生成的内容。

## 典型目录结构

```text
<root>/
├── <module>-api/           # 对外接口定义（DTO、接口声明）
├── <module>-service/       # 业务实现
│   └── src/main/java/com/<org>/<module>/
│       ├── controller/     # HTTP 控制器
│       ├── service/        # 业务服务接口与实现
│       ├── repository/     # 数据访问层
│       ├── domain/         # 领域模型（Entity、VO）
│       ├── dto/            # 数据传输对象
│       ├── config/         # 配置类
│       └── exception/      # 异常定义
└── <module>-start/         # 启动模块（含 application.yml）
```

## 包命名规则

- 顶层包：`com.<org>.<project>`
- 按职责分层，禁止跨层直接调用（Controller → Service → Repository）
- 禁止在 Controller 中直接操作 Repository

## 资源目录

- `src/main/resources/application.yml`：主配置
- `src/main/resources/application-<env>.yml`：环境配置
- `src/main/resources/mapper/`：MyBatis XML（如适用）
