# domains skills

本目录预留给“按能力域沉淀、且可被多个专家复用”的 skill。

当前不建议一开始就塞很多内容进来，只有在下面两种情况同时满足时再新增：

- 这个 skill 明显属于某个能力域，例如性能、安全、可观测性
- 这个 skill 会被多个专家复用，而不是只被一个专家调用

当前阶段建议优先继续使用：

- `common/` 里的通用 skill
- `profiles/react/` 和 `profiles/vue/` 里的技术栈 skill

后续如果需要，再在本目录下增加如：

- `performance/`
- `observability/`
- `security-a11y/`
