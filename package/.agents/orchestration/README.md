# 主代理协议目录

本目录用于存放 `task-orchestrator` 的内部协议、运行时载荷、桥接说明与配套模板。

目录约定：

- `roles/`：谁来做
- `skills/`：怎么做
- `flows/`：按什么顺序做
- `orchestration/`：主代理运行协议、载荷规范、桥接约定
- `registry/`：安装与运行时索引

放在这里的文件不是“专家角色”，而是给主代理、运行时适配层、抽取器和桥接层使用的支撑文档。

当前包含的文件类型：

- `*-spec.md`：协议与数据结构说明
- `*-payload.md`：桥接/适配载荷规范
- `*-template.md`：主代理输出模板
- `*-hooks.md`：运行时钩子与调用约定

新增主代理配套文件时，优先放在本目录，不再回写到 `.agents/roles/common/`。
