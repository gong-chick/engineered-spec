# 交互说明模板

当需求包含搜索、表单、弹窗、批量操作、复杂状态切换等交互时，可先按这份模板收口，再写入 `proposal / design / tasks`。

```md
## 交互说明

### 页面概述
- 目标：
- 入口：
- 关键用户路径：

### 布局与区域
- 主区域：
- 次区域：
- 响应式注意点：

### 关键交互
#### 搜索 / 筛选
- 字段：
- 查询时机：
- 重置行为：

#### 表单 / 弹窗
- 打开条件：
- 校验规则：
- 提交成功 / 失败反馈：

#### 列表 / 批量操作
- 批量选择：
- 空态 / 加载态 / 错误态：
- 分页与筛选联动：

### 数据与状态
- 数据来源：
- mock / 真实接口边界：
- 本地状态与全局状态分工：

### 验收关注点
- 用户是否能完成关键路径：
- 是否存在易错或易漏状态：
```
