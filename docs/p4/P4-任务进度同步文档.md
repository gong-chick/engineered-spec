### EventGateway 新增参数与方法

| 名称 | 类型 | 说明 |
|------|------|------|
| `options.throwOnWriteError` | boolean | 写入失败时是否抛出异常（默认 false） |
| `getLoadErrors()` | 方法 | 返回加载错误数组副本（type/message/timestamp/lineNumber/line） |
| `getWriteErrors()` | 方法 | 返回写入错误数组副本（type/message/timestamp） |
| `clear({ clearFile: true })` | 方法 | 同时清空内存和 NDJSON 文件 |

### RiskBoard 风险语义映射

| severity | riskLevel | rank |
|----------|-----------|------|
| info | low | 0 |
| warn | medium | 1 |
| error | high | 2 |
| blocking | critical | 3 |
| low/medium/high/critical | 保持原语义 | 0-3 |
| 其他 | low | 0 |
