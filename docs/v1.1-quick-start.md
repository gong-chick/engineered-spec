## 临时项目 Smoke Test

只允许写入 `/tmp/ai-spec-v1-1-smoke`：

```bash
AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js init /tmp/ai-spec-v1-1-smoke \
  --recommend --dry-run \
  --hub-url http://localhost:3000 \
  --visual-url http://172.20.10.4:18780

AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js init /tmp/ai-spec-v1-1-smoke \
  --recommend --yes \
  --hub-url http://localhost:3000 \
  --visual-url http://172.20.10.4:18780

AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js sync /tmp/ai-spec-v1-1-smoke \
  --hub-url http://localhost:3000

AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js check /tmp/ai-spec-v1-1-smoke
AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js guard assets /tmp/ai-spec-v1-1-smoke
```

## 真实项目 Dry-run 回归

真实业务项目只能执行：

```bash
AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js scan "<项目路径>" --explain
AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js scan "<项目路径>" --json
AI_SPEC_SKIP_LAUNCHER_SYNC=1 node bin/cli.js init "<项目路径>" --recommend --dry-run --hub-url http://localhost:3000
```

禁止对真实业务项目执行 `init --yes`。

## 隐私安全说明

V1.1 禁止上传：

1. 源码正文。
2. rawPrompt(原始提示词)。
3. rawResponse(原始响应)。
4. 绝对路径和 `/Users/` 路径。
5. apiKey、password、token、secret、`.env` 内容。

`changedFiles` 只能使用相对路径。
