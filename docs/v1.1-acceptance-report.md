## 15. 风险点

- Hub 和 Visual 真实服务依赖本地运行状态，演示前需要确认端口和服务进程。

- Visual 页面登录态可能影响现场演示，需要提前验证。
- warning(警告) 虽不阻断，但长期会降低后续回归定位效率。
