## P5.8 Asset Hub 企业级闭环加固（30 用例）

### 真实安装/升级/回滚专项（12 用例）

- install 写入真实文件到 projectRoot
- install 记录非空 installedFiles（含 source/target/checksum/action）
- install 记录真实 checksum（sha256 格式）
- install dryRun 不写文件
- install 资产不存在抛错
- upgrade 更新文件内容
- upgrade 记录 previousVersion
- upgrade metadata 包含 previousVersion
- rollback 恢复旧版本文件
- rollback version 为真实版本（非 'rollback'）
- lock 不出现 version: 'rollback'
- lock 包含 checksum 和 installedFiles

- metadata.rawPrompt 不得明文落盘
