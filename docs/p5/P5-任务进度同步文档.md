## 设计模式

| 模式 | 说明 |
|------|------|
| 工厂函数 | `createXxx(options)` 包装 `new Xxx(options)` |
| NDJSON 持久化 | fs.appendFileSync 逐行写入，loadFromFile 启动恢复，坏行容错跳过 |
| tombstone 记录 | 注销操作写入 `__deleted__` 标记记录，重启后不恢复 |
| 协调模式 | 高层模型（Manager/Installer/Lifecycle/Fork/Quality）协调多个基础模型 |
| barrel 导出 | `module.exports = { ...sub }` 统一导出 |
