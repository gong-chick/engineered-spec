### verifyChecksum 新语义

- `verifyChecksum(pkg)` 计算 `generatedFiles` 对应文件内容的包级 checksum
- 计算规则：对文件路径排序后，拼接 `相对路径 + "\n---AI_SPEC_FILE_CONTENT---\n" + 文件内容`，整体计算 sha256
- 使用 posix 路径分隔符保证跨平台一致性
- 返回 `{ ok, expected, actual, errors }`
- 文件缺失时返回 `ok: false` 并在 errors 中说明
