- 如果你需要先明确“这个项目当前要解决什么问题、目标用户是谁、交付边界和验收标准是什么”，先读 [需求说明文档](./需求说明文档.md)。
- 如果你当前主要目标是完成本次 `AI Coding(AI 编程)` 考核填报、整理提交物和确认 deadline(截止时间)，先读 [AI Coding考核提交说明](./AI%20Coding%E8%80%83%E6%A0%B8%E6%8F%90%E4%BA%A4%E8%AF%B4%E6%98%8E.md)。

- 如果你关注“第一次接触这套体系时为什么会觉得重，以及应该如何收束入口”，先读 [入口体验优化方案](./入口体验优化方案.md)。
- 如果你希望直接拿一份面向开发者、维护者和项目负责人的可执行用法，先读 [入口最佳实践指南](./入口最佳实践指南.md)。
- 如果你当前重点关注“为什么协议命令要切到用户侧全局入口、自动更新为什么不该落到项目目录、失败时怎么兜底”，先读 [全局Launcher与用户侧Runtime缓存机制说明](./全局Launcher与用户侧Runtime缓存机制说明.md)。
